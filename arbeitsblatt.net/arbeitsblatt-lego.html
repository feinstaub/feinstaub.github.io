Python mit dem NXT verbinden

$ cd ~/tmp/pynxt/

$ git clone https://github.com/eelviny/nxt-python

$ python setup.py install --user

import nxt -→ ok

https://github.com/Eelviny/nxt-python/wiki/Installation

sudo zypper install python3-usb

sudo groupadd lego

sudo usermod -a -G lego <username>

sudo touch /etc/udev/rules.d/70-lego.rules

ls -la /etc/udev/rules.d/70-lego.rules

sudo chmod u+w /etc/udev/rules.d/70-lego.rules

kdesu kwrite /etc/udev/rules.d/70-lego.rules

Save.

$ cd ~/tmp/pynxt/

git clone https://github.com/Eelviny/nxt-python-examples

checkout bytes branch from above repo! (siehe https://github.com/Eelviny/nxt-python/issues/72)
