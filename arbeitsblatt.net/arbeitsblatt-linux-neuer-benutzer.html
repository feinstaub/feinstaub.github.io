Neuen Benutzer anlegen

Übersicht
Inhalt

Das openSUSE-Image von osboxes.org hat einen Standard-Benutzer names "osboxes" (siehe http://www.osboxes.org/opensuse/).

Um das eigene System mit einem eigenen Benutzer zu betreiben, kann man einen eigenen Benutzer anlegen.

Es handelt sich dabei um einen system-eigenen Benutzer, der nur auf deinem Rechner (bzw. virtuellen Maschine) existiert. Insbesondere gibt keinen Abgleich mit anderen Computern im Internet.

mit YaST

  1. YaST aus dem K-Menü aufrufen.

  2. "Benutzer und Gruppen" aufrufen.

  3. Eigenen Benutzernamen (z. B. Vorname in Kleinbuchstaben oder ein Alias) und Passwort vergeben

  4. OK

  5. Mit diesem Benutzer ab sofort einloggen.
