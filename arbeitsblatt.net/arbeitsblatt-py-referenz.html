Einstieg auf openSUSE

  • python3 installieren, falls nicht schon vorhanden.

Gebe python --version ein. Wenn die Ausgabe nicht mit 3 beginnt, dann füge folgende Zeile ans Ende deiner ~/.bashrc ein: alias python=python3

  • Optional: Offline-Doku installieren: sudo zypper install python3-doc (bei Bedarf auch python3-doc-pdf)

    • Die Doku ist dann hier lokal verfügbar: file:///usr/share/doc/packages/python3/html/index.html

    • Ansonsten ist die Doku auch online zu finden: https://docs.python.org/3/

  • kate-Texteditor installieren und verschiedene Plugins aktivieren, unter anderem das Terminal-Plugin.

    • Speichern: Strg+S

    • Tools → Shortcuts → Focus Terminal: F4

    • Auskommentieren: Strg+D

    • Einkommentieren: Strg+Umschalt+D

    • Zeile nach oben/unten verschieben: Strg+Umschalt+Cursor hoch/runter

    • Zeile nach oben/unten kopieren: Strg+Alt+Cursor hoch/runter

  • Python-Shell (python-kurs.eu)

    • Ausloggen mit Strg+D

  • Interaktive Hilfe

    • Scripts ausführen und eingebaute Hilfe (python-kurs.eu)

    • help, type

    • ipython: ?, ??

    • objekt. TAB TAB

while-Schleife

i = 0
while i < 5:
    print(i)
    i += 1
# Ausgabe: 0, 1, 2, 3, 4
# Unendlich-Schleife:

i = 0
while True:
    print(i)
    i += 1
# Ausgabe: 0, 1, 2, 3, 4, 5, 6, 7, ...
Unterbrechen mit Strg+C. 
# break

i = 0
while True:
    print(i)
    if i == 4:
        break      # Vorzeitiger Abbruch
    i += 1
# Ausgabe: 0, 1, 2, 3, 4
# continue

i = 0
while i < 5:
    if i == 1 or i == 3:
        i += 1
        continue   # Rest des Schleifenblockes überspringen
    print(i)
    i += 1
# Ausgabe: 0, 2, 4

def print_plus(a, b):
    ergebnis = a + b
    print("a + b = {}".format(ergebnis))

print_plus(10, 5)
# a + b = 15


def my_plus(a, b):
    return a + b

print(my_plus(10, 5))
# 15
